Technical Specifications of the IRN and Digitally Signed QR CodeCryptographic IRN Hash Generation ProcessThe Invoice Reference Number (IRN) is a unique, system-generated identity of an e-invoice within the GST framework. The IRN is generated automatically by the IRP and is not a standard sequential billing number.The IRN is a 64-character hexadecimal hash created by applying the Secure Hash Algorithm 256-bit (SHA-256) cryptographic function to three specific parameters extracted from the invoice :Supplier GSTIN: The unique, 15-digit Goods and Services Tax Identification Number of the billing entity.Financial Year: The financial year in which the invoice is generated, formatted as YYYY-YY (e.g., 2025-26).Invoice Number: The original document number exactly as entered in the ERP, including special characters and alphabetic case styles.These three variables are concatenated into a single input string in a specific sequence, with the SHA-256 algorithm then applied to output the unique 64-character alphanumeric string.The mathematical operation is represented as:$$\text{Input String} = \text{Supplier's GSTIN} + \text{Financial Year (YYYY-YY)} + \text{Invoice Number}$$$$\text{IRN} = \text{SHA-256}(\text{Input String})$$This cryptographic method ensures that every invoice generated in a financial year receives a unique hash, making duplicate invoice numbers mathematically impossible and protecting the system from tampering.Anatomy of the Signed QR Code PayloadAfter successfully validating an invoice, the IRP returns a two-dimensional, digitally signed QR code containing key transaction details. This QR code enables offline verification of the invoice's authenticity by tax officers, buyers, and logistics operators on the go.The QR code is embedded with a compressed, digitally signed payload consisting of the following transaction parameters :Supplier GSTIN: The 15-character identifier of the billing supplier.Recipient GSTIN: The 15-character identifier of the registered purchasing entity.Document Number: The unique sequential invoice number assigned by the supplier.Document Type: The designated document classification, such as Tax Invoice, Debit Note, or Credit Note.Document Date: The official date of document issuance.Total Invoice Value: The gross monetary value of the transaction, including tax components.Number of Line Items: The total count of distinct product or service lines in the invoice.Main HSN: The Harmonized System of Nomenclature code of the highest-value item in the transaction.Invoice Reference Number (IRN): The 64-character cryptographic hash generated by the portal.IRN Generation Date: The timestamp recording when the invoice was successfully registered on the portal.Users can scan this QR code using the e-Invoice Verifier QR Code Mobile App or the GSTN e-Services App to instantly verify that the invoice is registered and has not been altered.
